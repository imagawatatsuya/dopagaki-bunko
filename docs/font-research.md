# Font Research Notes

## 2026-06-15 Dyslexia / BIZ UDGothic

- ディスレクシア対策としてのフォント変更は保留とする。
- 調査結果のみ記録する。

### BIZ UDGothic の位置づけ

- `BIZ UDGothic` は、確認した公式情報上ではディスレクシア専用フォントではない。
- 位置づけは、教育や業務文書向けの `UD` フォントであり、多くの利用者にとって読みやすいことを意図した書体である。
- したがって、可読性改善の候補にはなるが、ディスレクシア特化策として断定して採用するのは避ける。

### 青空文庫との相性

- `BIZ UDGothic` の公開情報には `Google Core glyph set` ベースでのリリースとある。
- そのため、日常的な日本語表示には有効でも、青空文庫に含まれうる旧字・異体字・稀少字まで単独で十分に網羅する前提は危険。
- 原文忠実性を重視する場合、`BIZ UDGothic` 単独依存は避けるべきである。

### iPhone 8 Chrome の表示環境

- iPhone 上の Chrome は、独自の日本語フォント環境ではなく、iOS の描画基盤とシステムフォールバック上で動作する。
- 実運用上の文字カバレッジは、単独フォントである `BIZ UDGothic` より、iPhone 標準の日本語表示環境のほうが広いとみなすのが妥当。

### 将来の方針候補

- フォント機能を再検討する場合は、目的を分ける。
- `読みやすさ重視モード`
- `原文忠実性重視モード`
- 1つのフォントで両立できる前提には立たない。
