# UIページ仕様

## ホームTL

目的：x.comの代替として縦スクロールで読む。

表示：

- 作品名
- 本文断片

表示しない：

- アイコン
- 投稿日時
- 操作アイコン
- エンゲージメント数

## 断片個別ページ

目的：断片をじっくり読み、操作する。

表示：

- 作品名
- 著者名
- 本文断片
- 前へ
- 次へ
- いいね
- しおり
- 引用として保存

## 作品ページ

目的：作品全体の進捗と断片一覧を見る。

表示：

- 作品名
- 著者名
- 読了進捗
- 現在位置の追従ヘッダ
- 断片一覧
- 続きから読む
- 本文サイズのその場調整
- 右下断片番号によるしおりトグル

## ライブラリ

目的：取り込んだ作品、しおり、引用を管理する。

## 検索

目的：青空文庫作品を探す、または取り込み済み作品を探す。

初期ではローカル検索のみでよい。

## 設定

目的：バックアップと入出力を管理する。
