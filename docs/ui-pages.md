# UIページ仕様

## ホームTL

目的：x.comの代替として縦スクロールで読む。

表示：

- 作品名
- 本文断片

表示しない：

- アイコン
- 投稿日時
- 操作アイコン
- エンゲージメント数

補足：

- ホームTLの `作品追加` カードを開くと、作品ページの先頭を表示する
- ホームTLの `最新しおり` と `いいね追加` のカードを開くと、作品ページ内の該当断片へ直接ジャンプする

## 断片個別ページ

目的：断片をじっくり読み、操作する。

表示：

- 作品名
- 著者名
- 本文断片
- 前へ
- 次へ
- いいね
- しおり
- 引用として保存

補足：

- しおりは同一作品内で常に1つだけ
- 新しい断片でしおりを付けると、古いしおりは置き換わる
- いいねは付箋として複数残せる

## 作品ページ

目的：作品全体の進捗と断片一覧、および現在のしおり位置を見る。

表示：

- 作品名
- 著者名
- 読了進捗
- 現在位置の追従ヘッダ
- 断片一覧
- しおり位置を開く
- 本文サイズのその場調整
- 右下断片番号による `なし → しおり → いいね → なし` の循環トグル

補足：

- `しおり` は輪郭主体で示す
- `いいね` は背景だけ淡いハート調で示す
- `しおり位置を開く` は単体ページではなく、作品ページ内の該当断片へ直接ジャンプする

## ライブラリ

目的：取り込んだ作品、作品ごとの最新しおり、いいね、引用を管理する。

## 検索

目的：青空文庫作品を探す、または取り込み済み作品を探す。

初期ではローカル検索のみでよい。

## 設定

目的：バックアップと入出力を管理する。
