# 青空文庫取り込みフロー

## 目的

青空文庫のルビありZIPを直接取り込み、作品アカウントと本文断片を生成する。

## 処理順

1. ZIP選択
2. ZIPサイズ検査
3. ZIP内ファイル一覧取得
4. txtファイル抽出
5. store/deflate判定
6. txt展開
7. Shift_JISデコード
8. ヘッダ・フッタ除去
9. ルビ変換
10. 圏点変換
11. HTML安全化
12. 断片化
13. プレビュー
14. 保存

## 本文範囲推定

青空文庫テキストには冒頭の注記説明と末尾の底本情報が含まれる。
自動判定は完全ではないため、保存前にプレビューで確認できるようにする。

## 失敗時表示

専門用語を避け、ユーザーが次に何をすればよいかわかる文にする。
