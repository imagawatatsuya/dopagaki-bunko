# 青空文庫取り込みフロー

## 目的

青空文庫のルビありZIPを直接取り込み、作品アカウントと本文断片を生成する。

## 処理順

1. ZIP選択
2. ZIPサイズ検査
3. ZIP内ファイル一覧取得
4. txtファイル抽出
5. store/deflate判定
6. txt展開
7. Shift_JISデコード
8. ヘッダ・フッタ除去
9. 外字注記の一部解決
10. ルビ変換
11. 圏点変換
12. HTML安全化
13. 断片化
14. プレビュー
15. 保存

現状の外字注記は、青空文庫の `※［＃「...」、U+....、...］` のように
Unicode 番号が明示されたものだけ実字へ置換する。

`※［＃「...」、第3/4水準...］` の面区点注記は、青空文庫公式の外字注記辞書と
JIS X 0213 の字形判断に基づく必要があるため、辞書ベースの検証済み対応を持つまでは
原文のまま残す。

## 本文範囲推定

青空文庫テキストには冒頭の注記説明と末尾の底本情報が含まれる。
自動判定は完全ではないため、保存前にプレビューで確認できるようにする。

## 失敗時表示

専門用語を避け、ユーザーが次に何をすればよいかわかる文にする。
