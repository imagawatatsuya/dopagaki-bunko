# セキュリティ方針

## ZIP

- ZIPサイズ上限を設ける
- 展開後サイズ上限を設ける
- txt以外を無視する
- パス情報を信用しない
- ZIP64、分割ZIP、パスワード付きZIPは拒否する

## HTML

青空文庫テキストはHTMLとして信用しない。
必ずエスケープしてから、許可するruby、rt、span等に変換する。

## GitHub Pages

GitHub Pagesで公開されるファイルに個人データを含めない。
.envや秘密情報をコミットしない。

## 依存

外部依存は原則禁止。
どうしても追加が必要な場合は、docs/dependencies.mdに理由とリスクを記録する。
